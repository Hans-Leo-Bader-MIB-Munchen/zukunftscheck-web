# Static Shell Checklist

Stand: v0.1

Vor Merge pruefen:

- [ ] Seite ist rein statisch.
- [ ] Keine Formulare vorhanden.
- [ ] Keine Upload-Funktion vorhanden.

- [ ] Keine externen Tracker oder Marketing-Skripte vorhanden.
- [ ] Robots-Meta steht auf noindex,nofollow.
- [ ] Texte behaupten keine externe Freigabe.
- [ ] Status der Seite ist als interner technischer Aufbau erkennbar.
