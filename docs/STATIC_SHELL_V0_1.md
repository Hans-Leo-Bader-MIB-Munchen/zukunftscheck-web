# ZS-WEB v0.1 Static Shell

## Zweck

Diese Version legt die erste rein statische Website-Shell fuer zukunftscheck.org an.

## Status

Interner technischer Aufbau. Keine oeffentliche Freigabe. Kein Produktivbetrieb.

## Enthalten

- statische Startseite `index.html`
- zentrale CSS-Datei `src/styles/main.css`
- Platzhalterordner fuer Komponenten, Seiten, Public Assets und Tests
- Checkliste fuer die Sichtpruefung

## Nicht enthalten

- keine Vorabfrage
- keine Formulare
- keine Uploads
- keine Datenerhebung
- keine Partnerweitergabe
- kein Tracking
- keine Vercel-Anbindung

## Naechster Schritt

Sichtpruefung des Pull Requests. Danach kann die statische Shell nach `main` gemerged werden. Erst danach folgt die Vorbereitung der Vercel-Staging-Anbindung.
